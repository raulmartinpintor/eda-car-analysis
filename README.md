# 🚗 EDA - Análisis de Dataset de Coches

## 📌 Descripción

Proyecto de análisis exploratorio de datos (EDA) sobre un dataset de automóviles obtenido de Kaggle.

El objetivo es analizar, limpiar y transformar los datos para extraer información útil mediante visualizaciones y análisis.

---

## 🛠️ Tecnologías utilizadas

- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  

---

## 📂 Estructura del proyecto

- data/
    - dataset_sucio_para_limpieza.csv
    - dataset_limpio.csv
- notebooks/
    - 1_eda_exploracion.ipynb
    - 2_eda_limpieza.ipynb
    - 3_eda_visualizacion.ipynb

- README.md
- requirements.txt

---

## 🔍 Análisis realizado

### 1. Exploración de datos
- Análisis inicial del dataset  
- Identificación de valores nulos, duplicados e inconsistencias  
- Revisión de tipos de datos y distribución  

### 2. Limpieza y transformación
- Eliminación de símbolos y formatos incorrectos  
- Conversión de variables a tipos numéricos  
- Normalización de categorías  
- Tratamiento de valores faltantes  

### 3. Visualización
- Histogramas para distribución de variables  
- Gráficos de barras para comparación de categorías  
- Boxplots para detección de outliers  
- Scatterplots para relaciones entre variables  

---

## 📊 Resultados

- Dataset limpio y estructurado listo para análisis  
- Identificación de patrones en variables como potencia, combustible y precio  
- Mejora de la calidad de los datos para futuros modelos  

---

## 🚀 Objetivo

Aplicar el proceso completo de análisis de datos:
- Exploración  
- Limpieza  
- Visualización
